# 🚀 APCS 太空探險課程 - Web UI

## 📋 項目說明

這是 APCS 太空探險課程的 Web UI 介面，提供沉浸式的學習體驗，與 FAE_official 保持一致的視覺風格。

## 🎨 設計特色

### 視覺風格
- **太空主題**：深空黑背景 + 3D 星空動畫
- **霓虹配色**：青色 (#00d9ff) + 紫色 (#a855f7)
- **玻璃態效果**：半透明背景 + 模糊效果
- **HUD 風格**：科技感邊框 + 發光效果

### 互動體驗
- **平滑滾動**：所有錨點連結都有平滑動畫
- **懸停效果**：卡片懸停時有提升和發光效果
- **進度追蹤**：自動記錄學習進度
- **響應式設計**：完美支援桌面和移動裝置

## 🚀 快速開始

### 方法 1：直接打開（推薦）
1. 用瀏覽器打開 `index.html`
2. 開始探索課程！

### 方法 2：本地伺服器
```bash
# 使用 Python 啟動簡單伺服器
cd web
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

然後訪問：http://localhost:8000

## 🔧 自定義配置

### 修改顏色主題
編輯 `styles/main.css` 中的 `:root` 變數：

```css
:root {
    --energy-cyan: #your-color;
    --energy-purple: #your-color;
}
```

### 修改星空效果
編輯 `scripts/starfield.js`：

```javascript
this.numStars = 200;  // 星星數量
this.speed = 0.5;     // 移動速度
```

### 添加新課程
1. 複製 `lessons/lesson-template.html`
2. 修改內容
3. 更新 `index.html` 中的課程連結
