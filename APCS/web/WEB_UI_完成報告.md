## 🎨 設計系統

### 顏色配色

```css
主色調：
- 深空黑：#0a0e27
- 深空藍：#1a1f3a
- 星雲藍：#2d4a7c

強調色：
- 能量青：#00d9ff （主要強調）
- 能量紫：#a855f7 （次要強調）
- 星光白：#e8f4f8 （文字）

功能色：
- 警告琥珀：#ffb800
- 危險紅：#ff3366
- 成功綠：#00ff88
```

### 視覺效果

1. **玻璃態效果**
   ```css
   background: rgba(26, 31, 58, 0.6);
   backdrop-filter: blur(10px);
   border: 1px solid rgba(0, 217, 255, 0.2);
   ```

2. **HUD 邊框**
   ```css
   border: 2px solid var(--energy-cyan);
   box-shadow: 
       0 0 10px rgba(0, 217, 255, 0.3),
       inset 0 0 10px rgba(0, 217, 255, 0.1);
   ```

3. **文字發光**
   ```css
   text-shadow: 
       0 0 20px rgba(0, 217, 255, 0.5),
       0 0 40px rgba(0, 217, 255, 0.3);
   ```

### 動畫系統

- **fade-in**：淡入效果
- **fade-in-up**：向上淡入
- **pulse-glow**：脈衝發光
- **float**：浮動效果
- **slide-up**：向上滑動
