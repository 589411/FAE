## 💡 技術亮點

### 1. 程式碼行號系統
```javascript
// 行號與程式碼在同一行，中間空一格
<span class="line-number">1</span> <span class="code-content">import sys</span>
```

### 2. 重點標示系統
```css
.code-line.highlight-key {
    background: rgba(0, 217, 255, 0.08);
    border-left: 3px solid var(--energy-cyan);
}
```

### 3. 自動檢測邏輯
```javascript
if (line.includes('import sys')) {
    highlightClass = 'highlight-key';
    annotation = '<span class="code-annotation key">高速 I/O</span>';
}
```
