# 🎨 程式碼重點標示系統

## 📋 功能說明

這個系統讓程式碼更容易閱讀和學習，透過顏色和標籤突出重點。

## 🎯 視覺效果

### 行號樣式
- **一般行號**：淡白色（rgba(255, 255, 255, 0.3)）
- **重點行號**：使用對應的主題色，字體加粗

### 重點行效果
每種重點行都有：
1. **彩色行號**
2. **左側彩色邊框**（3px）
3. **淺色背景**
4. **右側說明標籤**

## 🏷️ 標籤類型

### 1. 🔑 核心概念（青色）
- **顏色**：`#00d9ff`（能量青）
- **用途**：標示核心概念、關鍵語句
- **範例**：
  ```python
  import sys  # 🔑 匯入系統模組
  ```
- **自動檢測**：`import sys`、`sys.stdin.readline`

### 2. ⚡ 重要（紫色）
- **顏色**：`#a855f7`（能量紫）
- **用途**：重要邏輯、函式定義
- **範例**：
  ```python
  countdown = int(sys.stdin.readline())  # ⚡ 使用高速 I/O 讀取
  ```
- **自動檢測**：`def `（函式定義）

### 3. ⚠️ 注意（琥珀色）
- **顏色**：`#ffb800`（警告琥珀）
- **用途**：需要特別注意的地方、常見錯誤
- **範例**：
  ```python
  supplies = []  # ⚠️ 建立空列表儲存數據
  ```

### 4. ✅ 最佳實踐（綠色）
- **顏色**：`#00ff88`（成功綠）
- **用途**：推薦的寫法、內建函數使用
- **範例**：
  ```python
  total = sum(supplies)  # ✅ 使用 sum() 計算總和
  ```
- **自動檢測**：`return `（返回語句）

## 📝 使用方法

### 方法 1：手動標記（推薦）

在程式碼註解中加入特殊 emoji 標記：

```python
import sys  # 🔑 匯入系統模組

n = int(sys.stdin.readline())  # ⚡ 讀取第一行
supplies = []  # ⚠️ 建立空列表儲存數據
total = sum(supplies)  # ✅ 使用 sum() 計算總和
```

### 方法 2：自動檢測

系統會自動檢測以下關鍵字並標示：

- `import sys` → 🔑 核心概念
- `sys.stdin.readline` → 🔑 高速 I/O
- `def ` → ⚡ 函式定義
- `return ` → ✅ 最佳實踐

## 💡 設計理念

### 1. 顏色一致性
- 標籤顏色 = 行號顏色 = 邊框顏色
- 與 FAE_official 主題色系統一致

### 2. 視覺層次
- 一般行：低對比度（淡白色）
- 重點行：高對比度（主題色）
- 清晰的視覺分層

### 3. 閱讀輔助
- 行號與程式碼之間有明顯空格
- 重點行有背景色和邊框
- 右側標籤提供額外說明

## 📊 使用建議

### 何時使用各種標籤

#### 🔑 核心概念
- 課程的核心知識點
- 必須掌握的語法
- 關鍵的匯入語句

#### ⚡ 重要

- 函式定義
- 關鍵的運算步驟

#### ⚠️ 注意
- 容易出錯的地方
- 需要特別注意的細節
- 常見的陷阱

#### ✅ 最佳實踐
- 推薦的寫法
- 使用內建函數
- 高效的實現方式

### 標記密度建議
- 每 10 行程式碼標記 2-3 個重點
- 避免過度標記（會失去重點）
- 優先標記學習者容易忽略的地方
