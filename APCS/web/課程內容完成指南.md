## 🚀 快速完成方法

### 方法 1：手動複製貼上（推薦，最可靠）

#### 步驟：
1. 打開 `A2.html` 或 `A3.html`
2. 打開對應的 Markdown 文件
3. 找到需要替換的區塊
4. 將 Markdown 內容轉換為 HTML 並貼上

#### 範例：替換知識點說明

**找到 A2.html 中的這個區塊：**
```html
<!-- 知識點說明 -->
<section id="knowledge" class="content-section">
    <h2 class="section-heading">
        <span class="heading-icon">📚</span>
        <span>知識點說明</span>
    </h2>
    
    <!-- 這裡是需要替換的內容 -->
    <div class="knowledge-card glass">
        ...
    </div>
</section>
```

**從 Markdown 複製內容並轉換：**
- Markdown 的 `### 標題` → HTML 的 `<h3 class="knowledge-title">標題</h3>`
- Markdown 的 `**粗體**` → HTML 的 `<strong>粗體</strong>`
- Markdown 的程式碼區塊 → HTML 的 `<pre><code class="language-python">...</code></pre>`

## 💡 程式碼標記規則

在程式碼註解中加入特殊標記，系統會自動高亮：

```python
import sys  # 🔑 核心概念 - 青色
n = int(sys.stdin.readline())  # ⚡ 重要 - 紫色
supplies = []  # ⚠️ 注意 - 琥珀色
total = sum(supplies)  # ✅ 最佳實踐 - 綠色
```

### 標記使用建議
- **🔑 核心概念**：課程的核心知識點、必須掌握的語法
- **⚡ 重要**：重要的邏輯判斷、關鍵的運算步驟
- **⚠️ 注意**：容易出錯的地方、需要特別注意的細節
- **✅ 最佳實踐**：推薦的寫法、使用內建函數
