## 🚀 如何預覽

### 方法 1：直接打開（最簡單）

1. **預覽首頁**：
   ```bash
   # 在 Finder 中雙擊打開
   APCS/web/index.html
   ```

2. **預覽課程頁面**：
   ```bash
   # 在 Finder 中雙擊打開
   APCS/web/lessons/L1/A1.html
   ```

### 方法 2：本地伺服器（推薦）

```bash
# 進入 web 目錄
cd APCS/web

# 啟動 Python 伺服器
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000
```

然後在瀏覽器訪問：
- 首頁：http://localhost:8000/index.html
- 課程頁面：http://localhost:8000/lessons/L1/A1.html

## 🐛 常見問題

### Q: 星空背景不顯示？
**A:** 檢查瀏覽器是否支援 Canvas API，或嘗試重新整理頁面。

### Q: 程式碼沒有語法高亮？
**A:** 確保網路連線正常（需要載入 Prism.js CDN）。

### Q: 複製按鈕無效？
**A:** 檢查瀏覽器是否支援 Clipboard API，或使用 HTTPS/localhost。

### Q: 側邊欄導航不高亮？
**A:** 嘗試滾動頁面，高亮會根據當前章節自動更新。

### Q: 響應式布局異常？
**A:** 調整瀏覽器視窗大小，或使用開發者工具的裝置模擬器。
