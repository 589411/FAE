## 📚 每個單元包含什麼？

每個課程單元都有完整的結構：

## ⚡ 5 分鐘快速體驗

讓我們用一個簡單的例子開始：

### 範例：火箭倒數計時

```python
import sys

# 讀取倒數秒數
countdown = int(sys.stdin.readline())

# 倒數
for i in range(countdown, 0, -1):
    print(f"{i}...")

print("🚀 發射！")
```

**輸入**：
```
5
```

**輸出**：
```
5...
4...
3...
2...
1...
🚀 發射！
```

### 試試看！

1. 複製上面的程式碼
2. 在你的電腦上創建 `rocket.py`
3. 執行：`python rocket.py`
4. 輸入 `5` 並按 Enter

恭喜！你已經完成第一個太空任務！🎉

## 📖 學習方法建議

### ✅ 有效的學習方式

1. **動手寫**：不要只看，要親自打一遍
2. **理解原理**：知道「為什麼」比「怎麼做」更重要
3. **完成 Quiz**：每個單元的 Quiz 都要做
4. **定期複習**：學過的內容要常常回顧

### ❌ 無效的學習方式

1. 只看不寫
2. 跳過基礎單元
3. 死記硬背程式碼
4. 遇到困難就放棄

## 💡 常見問題

### Q: 我完全沒有程式基礎，可以學嗎？
**A:** 可以！課程從零開始，只要你願意學習。

### Q: 需要多久才能完成課程？
**A:** 依個人程度，大約 40-60 小時。建議每天學習 1-2 小時。

### Q: 我已經會 C++，還需要學這個嗎？
**A:** 如果你要用 Python 考 APCS，這個課程會幫助你快速掌握 Python 的特性和最佳實踐。

### Q: 課程涵蓋所有 APCS 題型嗎？
**A:** 涵蓋所有核心主題，但建議完成課程後還要刷題練習。

### Q: 我卡在某個單元怎麼辦？
**A:** 
1. 重新閱讀知識點說明
2. 逐行理解範例程式碼
3. 查看程式碼解說
4. 嘗試修改範例程式碼
5. 到線上社群求助
