## 🔧 技術細節

### 使用的 Python 特性
- sys.stdin.readline() - 高速輸入
- collections.deque - 高效佇列
- collections.defaultdict - 圖的表示
- enumerate() - 索引遍歷
- list comprehension - 列表生成式
- f-string - 字串格式化

### 強調的最佳實踐
- 使用 deque 而非 list.pop(0)
- 使用 join() 而非 += 拼接字串
- 使用內建函數而非手寫迴圈
- 使用記憶化優化遞迴
- 使用前綴和優化區間查詢
