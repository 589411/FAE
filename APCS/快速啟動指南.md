## 🚀 部署選項

### 選項 1：Cloudflare Pages（推薦）

1. 登入 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. Workers & Pages → Create application → Pages
3. 連接 GitHub repository
4. 設定：
   - **Production branch**: `main` 或 `feature/apcs-space-exploration`
   - **Build command**: 留空
   - **Build output directory**: `/`
   - **Root directory**: `APCS/web`
5. 部署完成！

### 選項 2：GitHub Pages

```bash
# 1. 確保在正確的分支
git checkout main  # 或你的目標分支

# 2. 推送到 GitHub
git add .
git commit -m "Add APCS space course"
git push origin main

# 3. 在 GitHub repo 設定中啟用 Pages
# Settings → Pages → Source: main branch → /APCS/web
```

### 選項 3：Netlify

1. 登入 [Netlify](https://www.netlify.com)
2. New site from Git
3. 選擇 repository
4. 設定：
   - **Base directory**: `APCS/web`
   - **Build command**: 留空
   - **Publish directory**: `APCS/web`
5. 部署！

## 🔧 故障排除

### 問題 1：本地預覽時樣式不顯示

**原因**：相對路徑問題

**解決**：確保從 `web/` 目錄啟動伺服器

### 問題 2：程式碼高亮不正常

**原因**：Prism.js 載入失敗

**解決**：檢查網路連線，Prism.js 從 CDN 載入

### 問題 3：課程頁面 404

**原因**：檔案路徑錯誤

**解決**：執行 `verify_lessons.py` 檢查所有文件
