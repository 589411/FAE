# 太陽能電池模擬器 - 教學指南

## 前頁：背景介紹

### 為什麼要學習太陽能發電？
太陽能是地球上最豐富的可再生能源之一。隨著全球對清潔能源的需求增加，了解太陽能發電的原理變得越來越重要。本模擬器旨在幫助學習者理解太陽能發電系統的基本工作原理。

### 學習目標
1. 理解太陽能板的工作原理
2. 了解太陽能發電效率與太陽角度的關係
3. 掌握電池儲能和用電管理的概念
4. 認識家用電器的用電特性

### 預備知識
- 基本的物理學概念
- 簡單的電學知識
- 角度和幾何的基礎概念
